# Formal Verification Exercises (Dafny)
My solutions to the exercises from the [University of Michigan's EECS 498: Formal Verification](https://glados-michigan.github.io/verification-class/fall2024/) course.

## Overview
This repository covers the fundamentals of formal verification using the **Dafny** language, progressing from basic functional correctness to proving safety and refinement of distributed systems.
